Add New Tape Storage

Here you can define a new tape library based backup storage.

Storage name - name of your new backup storage. It has to be unique in your system.

Description - any description of your backup storage.

Existing Storage - select already defined storage - Bacula Storage Daemon.

Available Libraries - select a tape library you want to add.

Before you can add the selected tape library IBAdmin has to detect its configuration. This can take some time and the detection progress will be shown here.

Departments - departments name for the storage; you can select multiple departments.

When you add a new storage you need to restart Storage Daemon manually for fully use of this storage.

You have to label tapes in new library before you can use them for backup jobs.
