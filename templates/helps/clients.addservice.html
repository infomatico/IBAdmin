New Cluster Service Backup Client

Here you can define a new cluster service backup client. Cluster service backup client is always associated with a cluster service floating IP address. All Cluster Node clients and Cluster Service clients of the same cluster name share some of its properities.

Service name - name of your new backup client - Cluster Service. It has to be unique in your system.

Description - any description of your backup client.

Cluster name - name of the existing cluster component for a cluster service client.

Address - IP address or FQDN of cluster service client. It has to be a valid (resolved) address.

Create default backup Job - when checked a defaut Job for the Client will be created together with a Client.

Default Job will have the name <client_name>-DefaultJob.
