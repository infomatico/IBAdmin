Files Job Advanced

Here you can edit advanced properities of files backup job.

Run Before Job - the command which is executed on Client before backup job. Full path expected.

Run After Job - the command which is executed on Client after a backup job finishes. Full path expected.

Job enabled - you can enable (on) or disable (off) the job execution.

Client deduplication - when enabled (on) deduplication job is performed on Client, when disabled (off) deduplication job is performed on Storage only.
