Edit Backup Client Alias

Here you can edit backup client Alias. Backup client Alias is like a symbolic link or CNAME which points to standard Backup Client.

Alias name - name of your backup client Alias you edit.

Description - any description of your backup client.

Alias to - name of the existing Backup Client (standalone or cluster node) to point to.

Department - department name for the client.
