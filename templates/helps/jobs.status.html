Job Status

Here you can view all details regarding your running JobId.

You can always refresh the view using Refresh button ().

Processing - Display currently processing file (trimed name).

Progress - Display estimated job progres based on previous jobs. For first job in history display Unknown.

Files examined - Display a number of files checked during backup.

Files backup - Display a number of files already saved by job.

Read Bytes - Display the size of data read by job during backup.

You can Cancel , Stop or Comment currently running JobId.

Use to view a Statistics of the Job or go stright to the Job Info page with .

Job Log - current log from execution of this JobId.

Data Location - information about storage location of this JobId.
