Deduplication Storage Engine Info

Here you can view all details about Deduplication Storage Enfine.

You can always refresh the view using Refresh button ().

Space size

Saved data size - space size for the all data stored without deduplication.

Deduplication Ratio - space save factor for using deduplication.

Disk Space Used - space size of all used data in containers.

Disk Space Allocated - disk space size allocated for containers.

Chunks stats

Reference count - number of chunks referenced by stored data.

Chunks Used - number of chunks used in containers.

Chunks Allocated - number of all allocated chunks in containers.

Status

DDE Errors - number of deduplication engine errors.

Containers Errors - number of chunks encountered errors.

Vacuum - date of the last run vacuum process.

It could be Never if no vacuum process was running till now or Running for currently running vacuum process.

Deduplication containers - display a stats for all deduplication containers listed by container block size.

1k - 65k - container block size.

Disk space allocated - disk space size allocated for this container.

Space used - space size used in container.

Used - space size used to space size allocated ratio for container.
