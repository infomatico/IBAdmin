Storage Status

Here you can view all details about Storage.

You can always refresh the view using Refresh button ().

You can Edit , Display detailed information about Deduplication Engine of your Storage or go straight to the Storage Info page with .

Deduplication Engine status is available for deduplication enabled storage only.

Devices Status - display a list of defined storage devices with some online information about each one.

Running - device is busy running a backup job.

Idle - device is idle, no job running on this device.

Disabled - device is temporary disabled, no job will be running on this device in the future. The Disabled status will be valid until manually enabled or Storage Daemon restart.

Display detailed information about any selected Storage Device.
