Edit Disk Storage

Here you can edit disk based backup storage. It is a standard file based storage.

Storage name - name of your backup storage you edit.

Description - any description of your backup storage.

Archive directory - location for the Volumes archive. All backups will be stored in this location.

Archive directory location must exist on local storage and has to be writable by bacula user. You can't make successful backup if Storage will be unable to write to this location.

If you have some Volumes at your old Archive directory you have to move these Volumes to new location unless you will be unable to make restores. It requires a manual Storage Daemon restart for fully use of this storage.
