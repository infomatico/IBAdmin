New Backup Client Alias

Here you can define a new backup client Alias. Backup client Alias is like a symbolic link or CNAME which points to standard Backup Client.

Alias name - name of your new backup client - Alias. It has to be unique in your system.

Description - any description of your backup client.

Alias to - name of the existing Backup Client (standalone or cluster node) to point to.

Create default backup Job - when checked a defaut Job for the Client will be created together with a Client.

Default Job will have the name <client_name>-DefaultJob.
