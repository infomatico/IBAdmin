Client Info

Here you can view all details and manage your backup client.

You can Check online Status , Define new Files Job , Restore , Edit and Delete the Backup Client.

You cannot delete a Client if it has a running Jobs. Deleting a Client will delete all defined Jobs and jobs history.

When display information about Virtual Host backup client (VMware, Proxmox, etc.) then you can Define New Guest VM Job instead.

You can download File Daemon config file for Standard and Cluster Node backup clients with download button

You can download vSphere Plugin config file for VMware Proxy client with dedicated download button
